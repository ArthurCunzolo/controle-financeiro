# 💰 FinControl — Sistema de Controle Financeiro Pessoal

O **FinControl** é um projeto pessoal desenvolvido com o objetivo de auxiliar no **controle financeiro pessoal**, permitindo o acompanhamento de contas, pagamentos, dívidas renegociadas e uma visão geral das finanças através de um **dashboard intuitivo**.

A aplicação busca fornecer uma **visão clara da situação financeira**, ajudando o usuário a organizar seus compromissos financeiros, evitar atrasos e acompanhar seus gastos de forma simples e eficiente.

---

# 📊 Funcionalidades

## 📈 Dashboard Financeiro

O sistema possui um **painel principal (dashboard)** que apresenta um resumo rápido da situação financeira do usuário.

No dashboard é possível visualizar:

* **Despesas do mês**
* **Pagamentos realizados**
* **Contas pendentes**
* **Contas em atraso**
* **Dívidas renegociadas**

Essas informações ajudam a ter uma **visão rápida da saúde financeira**.

---

## 🧾 Gestão de Contas

O sistema permite cadastrar e acompanhar contas a pagar.

Principais funcionalidades:

* Cadastro de novas contas
* Definição de valores
* Definição de datas de vencimento
* Identificação de contas pendentes
* Identificação de contas vencidas

---

## 💳 Registro de Pagamentos

A plataforma permite registrar pagamentos realizados, mantendo um histórico financeiro organizado.

Formas de pagamento suportadas:

* **PIX**
* **Cartão**
* **Boleto**

Isso permite acompanhar **como cada despesa foi paga**.

---

## 🔄 Renegociação de Dívidas

O sistema também possui uma área para controle de **dívidas renegociadas**, permitindo acompanhar acordos financeiros.

Nesta seção é possível:

* Registrar renegociações
* Acompanhar renegociações ativas
* Controlar valores renegociados

---

## 📅 Calendário Financeiro

O sistema conta com uma visualização em **calendário**, facilitando o acompanhamento de vencimentos de contas e compromissos financeiros futuros.

Isso ajuda o usuário a **evitar atrasos e organizar melhor seus pagamentos**.

---

## 📤 Exportação de Dados

A aplicação também permite **exportar dados financeiros**, possibilitando análises externas ou backups das informações.

---

# 🖥️ Interface

O sistema foi desenvolvido com foco em **simplicidade e usabilidade**, contendo:

* Dashboard financeiro
* Menu lateral de navegação
* Visualização de despesas por categoria
* Estatísticas por forma de pagamento
* Suporte a **modo escuro (Dark Mode)**
